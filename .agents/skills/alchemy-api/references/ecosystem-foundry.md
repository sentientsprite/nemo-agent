# Foundry

## Summary
Foundry is a fast, Rust-based Ethereum toolchain for testing and deployment.

## Why It Pairs Well With Alchemy
- Use Alchemy RPC URLs for `forge` tests, scripts, and deployments.

## Quick Setup
```bash
export ETH_RPC_URL="https://eth-mainnet.g.alchemy.com/v2/$ALCHEMY_API_KEY"
```
