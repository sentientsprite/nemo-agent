## Summary
Popular open-source libraries that pair well with Alchemy when building crypto apps. This section is intentionally concise.

## References (Recommended Order)

### EVM Libraries
1. [ecosystem-viem.md](ecosystem-viem.md) - Modern TypeScript Ethereum client with strong types.
2. [ecosystem-ethers.md](ecosystem-ethers.md) - Widely used Ethereum library for wallets and contracts.
3. [ecosystem-web3js.md](ecosystem-web3js.md) - Long-standing Ethereum JavaScript library.
4. [ecosystem-wagmi.md](ecosystem-wagmi.md) - React hooks for Ethereum with wallet connectors.
5. [ecosystem-rainbowkit.md](ecosystem-rainbowkit.md) - Wallet connection UI kit.

### EVM Development Frameworks
6. [ecosystem-hardhat.md](ecosystem-hardhat.md) - Ethereum development environment.
7. [ecosystem-foundry.md](ecosystem-foundry.md) - Rust-based Ethereum toolchain.

### Solana Libraries
8. [ecosystem-solana-web3js.md](ecosystem-solana-web3js.md) - Primary Solana JavaScript library.
9. [ecosystem-spl-token.md](ecosystem-spl-token.md) - Solana token program and management tools.
10. [ecosystem-anchor.md](ecosystem-anchor.md) - Solana framework for building programs.
11. [ecosystem-metaplex.md](ecosystem-metaplex.md) - Solana NFT standards and tooling.
