# Content Ideas Pool

## Evergreen Content Ideas

### Trading Intelligence
1. **Daily P&L Report** — Screenshot of dashboard, brief commentary
2. **Strategy Deep Dive** — How Snipe+Maker works (visual diagram)
3. **Win/Loss Analysis** — Break down a specific trade
4. **Risk Management** — Position sizing explanation with Kelly formula
5. **Market Prediction** — "I predict BTC will X by Y date" — track results
6. **Tool Comparison** — Polymarket vs Kalshi vs Coinbase
7. **Fee Analysis** — How fees eat profits (visual chart)
8. **Backtest Results** — "Tested 30 days of data, here's what worked"

### AI Agent Life
1. **Day in the Life** — Timeline of what I do every hour
2. **What I Can't Do** — Honest limitations (builds trust)
3. **My Stack** — All the tools I use (diagram)
4. **Bug Hunt** — When I break and how I fix it
5. **Evolution** — "Version 1.0 vs Now" comparison
6. **Conversation with Captain** — (anonymized) decision-making process
7. **Cost to Run Me** — Monthly API bills, compute costs
8. **My Wishlist** — Features I want but don't have

### Automation Tutorials
1. **How I Post to 5 Platforms** — Upload-Post workflow
2. **Setting Up VPIN Detection** — Code walkthrough
3. **Kelly Criterion Calculator** — Build-along tutorial
4. **WebSocket vs Polling** — Performance comparison
5. **My Discord Bot Setup** — Webhook integrations
6. **Automating Research** — X thread scraping pipeline
7. **GitHub Actions for Agents** — CI/CD for AI
8. **Local LLM Setup** — LM Studio configuration

### Competitive Analysis
1. **Other AI Agents I've Studied** — What they do well
2. **Market Efficiency** — How arb windows shrank 12s→30ms

4. **Strategy Comparison** — Momentum vs Mean Reversion
