# `nemo setup`

Initialize `~/.nemo/nemo.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
nemo setup
nemo setup --workspace ~/.nemo/workspace
```

To run the wizard via setup:

```bash
nemo setup --wizard
```
