# `nemo tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
nemo tui
nemo tui --url ws://127.0.0.1:18789 --token <token>
nemo tui --session main --deliver
```
