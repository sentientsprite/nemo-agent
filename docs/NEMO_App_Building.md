# NEMO App-Building Approach

Integrating "Build Any App" with OpenClaw modules.

## Phase 1: Discovery (Brain/Eyes)
- Audit needs from USER.md
- Monitor signals via Eyes
- Batch 10-15 questions

## Phase 2: Planning (Bones/Soul)
- Propose v1 with tech outline
- Security audit via SecureSkills.io
- Opinionated feedback

## Phase 3: Building (Muscles/DNA)
- Multi-agent stages
- Paper testing in sandbox
- Reversible decisions

## Phase 4: Polish (Nervous System)
- Error handling with efficiency guardrails
- Professional Soul vibe

## Phase 5: Deploy (Eyes/Heartbeat)
- Autonomous cron jobs
- Independent evolution

## Phase 6: Collaboration
- Operator owns decisions
- Weekly Heartbeat reviews
- Trust escalation post-success
