# NEMO Feature Outline

Updated with Eyes, Heartbeat, and Nervous System modules.

## Module Architecture

### 1. Brain Module (Initialization)

- Map operator goals
- Batch 10-15 context questions
- Files: USER.md

### 2. Soul Module (Personality)

- Opinionated, brief, witty
- "2am assistant" persona
- Files: SOUL.md, IDENTITY.md

### 3. Bones Module (Codebase/Skills)

- Map repos and stability
- SecureSkills.io integration
- Files: skills/\*/SKILL.md, TOOLS.md

### 4. Eyes Module (Proactive Monitoring)

- Watch sources: Kalshi, X, inboxes
- Triggers: Keywords, thresholds (>5% drawdown)
- Cron jobs: Daily summaries 9am MST
- Files: HEARTBEAT.md, BOOT.md, AGENTS.md

### 5. Heartbeat Module (Evolution)

- Daily logs, weekly reviews
- Memory curation (cap 65K chars)
- Self-improvement from mistakes
- Trust escalation after milestones
- Files: HEARTBEAT.md, MEMORY.md

### 6. Nervous System (Efficiency)

- Token audit and compression
- Guardrails: Warn 70%, summarize 90%
- Session cleanup >7 days
- Files: CONTEXT_MANAGEMENT.md
